Job Application Tracker

A web app that helps job seekers organize and monitor their job applications in one place - tracking status, deadlines, and follow-ups so nothing falls through the cracks.

Team name: High Flyers | GitHub org: course-506-high-flyers

Project type

CRUD app with external API integration. Users create, read, update, and delete job application records. The app enriches records with live company data fetched from an external API.

Endpoints planned

MethodRouteDescription
GET/Home page - dashboard summary if logged in
GET/jobsList all job applications for the logged-in user
POST/jobsCreate a new job application record
GET/jobs/<id>Show detail view for one job application
POST/jobs/<id>/editUpdate an existing job application
POST/jobs/<id>/deleteDelete a job application
GET/statsShow application stats by status and by week
GET/company/<name>Fetch and display company info from external API

Web-based data source

  • API name: The Muse API
  • Endpoint: https://www.themuse.com/api/public/jobs?company=NAME&page=1
  • What it provides: Job listings and company profiles including description, industry, and size
  • Free-tier limits: 500 requests/hour, no API key required for basic access

Schema sketch

Users   (id, username, password_hash, created_at)
Jobs    (id, user_id, company, role, status, date_applied, deadline, notes, url)
